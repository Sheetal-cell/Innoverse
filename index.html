Project Idea: NeuroEase ‚Äì An Inclusive Web App for Neurodiverse Individuals

NeuroEase is a web-based platform designed to support neurodiverse individuals by providing personalized assistance in education, employment, and daily life. It integrates accessibility features, AI-driven customization, and Google tools to enhance inclusivity.

Key Features

  • Customizable UI for Sensory Preferences: Adjustable color schemes, dyslexic-friendly fonts, sound/motion control.
  • AI-Powered Smart Assistant (Gemini API + Vertex AI): Task guidance, AI explanations, adaptive learning support.
  • Task Management & Executive Functioning Support: To-do lists, visual schedule planner, Pomodoro timer.
  • Job and Skill Matching Platform: AI-powered job suggestions, resume assistance, employer resources.
  • Emotion and Stress Management: Mood tracking, mindfulness exercises, text-to-speech check-ins.
  • Community and Support Forum: Safe space, Q&A sessions, mentorship programs, resource hub.

Tech Stack

Frontend: HTML, CSS, JavaScript

APIs & Tools: Google Gemini API (AI chatbot), Vertex AI, IDX, Firebase

Accessibility Features: ARIA roles, keyboard navigation, screen reader support
